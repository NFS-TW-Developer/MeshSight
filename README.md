# MeshSight

## 簡介
本倉儲存放了 [MeshSight-APP-Flutter](https://github.com/edwinyoo44/MeshSight-APP-Flutter) 的 API 介面，使用 Python + FastAPI 開發。並利用 PostgreSQL 進行資料管理。整個系統已整合至 Docker Compose，以實現便捷的容器化部署，確保 API 的高效運行與擴展性。
